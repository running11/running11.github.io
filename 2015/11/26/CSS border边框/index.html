CSS border边框

CSS border边框是可以控制一个元素边框的样式和颜色。

1、border-color:black;边框颜色

有默认值：默认颜色值是字体颜色，默认字体颜色是黑色。

属性值可以为英语代码如：red,blue等。

可以为16进制颜色码如：#ffffff,#000000等。

可以为RGB格式如：RGB(255,182,192)等。

可以分方向设置：

border-left-color:;边框左边颜色

border-right-color:;边框右边颜色

border-top-color:;边框上面颜色

border-bottom-color:;边框底部颜色

2、border-width:3px;边框宽度

有默认值：默认值是3px的宽度。数值不能为负。

可以分方向设置：

border-left-width:;边框左边宽度

border-right-width:;边框右边宽度

border-top-width:;边框上面宽度

border-bottom-width:;边框底部宽度

3、border-style:solid/dotted/dashed;边框样式

没有默认值,solid为实线,dotted点状线,dashed虚线

可以分方向设置：

border-left-style:;边框左边样式

border-right-style:;边框右边样式

border-top-style:;边框上面样式

border-bottom-style:;边框底部样式

4、border复合写法：

border:solid;一个值的时候，border-color、border-width是默认值

border:1px solid;border-color是默认值

border:1px red solid;
